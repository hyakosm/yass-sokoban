# Yass-Sokoban

Un petit jeu de Sokoban pour DOS, compatible 286 et plus.

![Illustration du jeu](https://hyakosm.net/images_portfolio/sokodos.jpg)

## Utilisation

- `F1` : recommencer le niveau
- `F2` : niveau précédent
- `F3` : niveau suivant
- `Échap` : quitter

## Compilation

Nécessite le compilateur NASM.

```nasm main.asm -o main.com```
